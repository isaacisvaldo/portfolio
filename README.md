## Running Locally

This application requires Node.js v18.13+.

```bash
npm install -g @angular/cli
git clone https://github.com/isaacisvaldo/portfolio
cd portfolio
npm install
```

```bash
 ng s
```

Wait to compile and go to http://localhost:4200 after compile finish
